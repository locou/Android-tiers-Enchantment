===Enchantment===

Sprinkle some fairy dust onto some of the gfx

Enchants the following *Things*:
* Friendly android base
* Hostile android base
* Battery gizmo
* Skill gizmo
* Skymind gizmo
* Upload gizmo
* Overloead gizmo


Can be added to or removed from a save game.
